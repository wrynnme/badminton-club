# Changelog — ก๊วนแบด (kuanbad)

บันทึก "มีอะไรใหม่ / เปลี่ยนอะไร" ในแต่ละรอบที่ขึ้น production — สำหรับเจ้าของก๊วนและผู้เล่น

> **รูปแบบ:** รุ่นใหม่อยู่บนสุด · หัวข้อ = `vX.Y.Z — วันที่ขึ้น production` · จัดกลุ่ม ✨ ใหม่ / 🔧 ปรับปรุง / 🐞 แก้บั๊ก
> **เลข version (semver):** ฟีเจอร์ใหม่ = +minor · แก้บั๊กล้วน = +patch · เลขล่าสุดต้องตรงกับ `package.json` และ `src/lib/changelog.ts` (`CURRENT_VERSION`)
> **วิธีเพิ่ม:** ทุกครั้งที่ขึ้น production (merge `develop → master`) เพิ่มหัวข้อรุ่นใหม่ไว้บนสุด + bump version
> **หมายเหตุ:** บันทึกภายในของ dev อยู่ที่ `spec.md` + `bug.md` — ไฟล์นี้คู่กับหน้า `/whats-new` (source = `src/lib/changelog.ts`)

## v0.43.0 — 2026-07-15

### ✨ ใหม่

- หน้าก๊วนถาวร — /clubs เปลี่ยนเป็นรายชื่อก๊วน มีหน้าแรกของก๊วน (ภาพรวม · สมาชิก · ตั้งค่า) ส่วนนัดแต่ละครั้งย้ายไปอยู่ใต้ก๊วน
- ปุ่ม "จัดก๊วน" — เปิดนัดใหม่คลิกเดียว: ดึงขาประจำลงรายชื่อ พร้อม level และคู่ล็อคประจำอัตโนมัติ ตามค่าเริ่มต้นนัดที่ตั้งไว้
- ทะเบียนสมาชิกก๊วน — เพิ่ม/แก้ชื่อ/ตั้ง level/ขาประจำ/คู่ประจำ รวมถึงสมาชิกแบบชื่ออย่างเดียว (ยังไม่ผูก LINE) และปุ่มรีเซ็ต level ทั้งก๊วน
- ก๊วนเฉพาะกิจ — สร้างนัดเล่นครั้งเดียวโดยไม่ต้องตั้งก๊วนถาวร และอัปเกรดเป็นก๊วนถาวรภายหลังได้
- เก็บก๊วนเข้าคลัง (archive) พร้อมกู้คืนได้จากหน้ารายชื่อก๊วน

### 🔧 ปรับปรุง

- ค่าเริ่มต้นนัด (สนาม เวลา ค่าใช้จ่าย การหาร สนามที่ใช้ และตั้งค่าคิวเต็มชุด) แก้ได้ในหน้าตั้งค่าก๊วน + ปุ่มดึงค่าจากนัดปัจจุบัน — ระบบ preset เดิมถูกถอดออก
- ลิงก์เข้าร่วม/ผูกกลุ่ม LINE และคิวรอจับคู่ ย้ายมาอยู่หน้าตั้งค่าก๊วนถาวร (จุดเดียวจบ)
- URL เดิมของนัดยังใช้ได้ — เด้งไปที่อยู่ใหม่ใต้ก๊วนอัตโนมัติ
- ช่องคำสั่งผูกก๊วน LINE ไม่มี scrollbar แล้ว (ข้อความยาวตัดบรรทัดแทน)

### 🐞 แก้บั๊ก

- ลบนัดเก่าไม่ทำให้คำขอเข้าร่วมก๊วนที่ค้างอยู่หายอีกต่อไป
- แก้ level ผู้เล่นที่เป็นสมาชิกในนัด จะอัปเดต level ประจำตัวสมาชิกให้ด้วย (ติดไปนัดถัดไป)

## v0.34.0 — 2026-07-13

### ✨ ใหม่

- เชื่อมบัญชี LINE กับก๊วนได้เองในกลุ่มแล้ว 🎉 — ไม่ต้องเปิดลิงก์เชิญหรือรอแอดมินจับคู่ให้อีกต่อไป
  - **วิธีใช้:** ในกลุ่ม LINE ที่ผูกก๊วนไว้ ให้แท็กบอท (พิมพ์ @ แล้วเลือกชื่อบอทของก๊วน) เว้นวรรค แล้วพิมพ์คำว่า "เชื่อมไลน์" ตามด้วยชื่อของคุณที่อยู่ในโพย เช่น `@ก๊วนแบด เชื่อมไลน์ โจ้`

  - หมายเหตุ: ต้องแท็กบอทเสมอระบบถึงจะรับคำสั่ง และใช้ได้เฉพาะในกลุ่มที่แอดมินผูกก๊วนไว้แล้วเท่านั้น

- **บิลไลน์เป็นสลิปสวยแทน QR เปล่า** — ทั้งบิลกลุ่มและบิลรายคน (DM) เปลี่ยนจาก QR เปล่า/การ์ดเขียวธรรมดา มาเป็น "สลิป" เต็มใบตามที่ตั้งค่าไว้ในหน้าปรับแต่งใบเสร็จ (ธีมสี โลโก้ก๊วน รายการค่าใช้จ่าย บัญชีธนาคาร ข้อความท้ายบิล) — บิลกลุ่มแสดงเฉพาะยอดรวม + QR (เพราะคนที่ยอดเท่ากันอาจมีรายละเอียดค่าใช้จ่ายต่างกัน) ส่วนบิลรายคนแสดงชื่อและรายการครบ ระหว่างสร้างสลิปปุ่มจะบอกความคืบหน้า "กำลังสร้างสลิป X/N" และถ้าบางใบสร้างไม่สำเร็จจะข้ามพร้อมแจ้งจำนวนให้ทราบ

## v0.9.0 — 2026-06-21

### ✨ ใหม่
- **ตั้งค่ายืนยันสลิปจ่ายเงินได้รายก๊วน** — แต่ละก๊วนเลือกเองได้ว่าจะตรวจสลิปแบบไหน
  - **ยืนยันเอง** (ค่าเริ่มต้น) — ผู้เล่นส่งสลิป แล้วเจ้าของก๊วนกดยืนยัน

  - กุญแจเก็บแบบปลอดภัย ไม่แสดงค่าจริงในหน้าเว็บ (โชว์แค่ว่า "ตั้งแล้ว")
- **ล็อกอินอัตโนมัติเมื่อเปิดผ่านแอป LINE** — เปิดเว็บในแอป LINE แล้วเข้าสู่ระบบเป็นบัญชี LINE นั้นทันที ไม่ต้องกดปุ่มล็อกอิน (ถ้าเปิดนอก LINE ใช้ปุ่มล็อกอินเดิมตามปกติ)

### 🔧 ปรับปรุง
- ยกเลิก "คีย์กลาง" ตรวจสลิปแบบเดิม เปลี่ยนเป็นกุญแจของแต่ละก๊วน — ตรวจแม่นยำขึ้นเพราะผูกกับบัญชีรับเงินของก๊วนเอง
